## For developement

Requirements: node >= v18.3.0, npm >= 8.16.0, python >=3.7.0

### Client

For installation,
```
cd client
npm install
```

For running the react client
```
vite
```

### Server
For installation,
```
cd server
pip install -r requirements.txt
```

For running the flask server

```
python3 main.py 
```
